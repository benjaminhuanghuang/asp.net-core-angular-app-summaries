
## create component
```
    cd src/app/components/
   
    ng g c books -m app
    ng g c delete-book -m app
    ng g c new-book -m app
    ng g c show-book -m app
    ng g c update-book -m app
```

## Create services
```
    cd src/app/services/
   
    ng g s book
```
Add service to providers array
